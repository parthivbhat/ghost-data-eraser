# Ghost Data Eraser 👻

AI system that detects online services you signed up for by scanning Gmail and identifying account creation emails.

This helps users understand their **digital footprint** and manage privacy.

---

## 🚀 Features

• Connects to Gmail using Gmail API  
• Detects signup emails automatically  
• Uses Machine Learning to classify account creation emails  
• Identifies services like LinkedIn, NVIDIA, NPTEL etc  
• Helps track digital accounts created over time  

---

## 🧠 Tech Stack

Python  
Gmail API  
Scikit-learn  
TF-IDF Vectorization  
Machine Learning (Text Classification)

---

## ⚙️ How It Works

1. Gmail API fetches email subjects and senders  
2. Dataset is created from inbox emails  
3. Emails are labeled (signup / non-signup)  
4. ML model is trained using TF-IDF + Logistic Regression  
5. Real-time detector identifies signup services
